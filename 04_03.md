# Задания 1

## Рассматривается множество целых чисел, принадлежащих отрезку [1012; 9638], которые делятся на 7 и оканчиваются на 1. Найдите сумму таких чисел. 

###
```
s = 0
for i in range (1012, 9639):
    if (i % 7 == 0) and (i % 10 == 1):
        s += i
print(s)
```
## Для самопроверки: программа должна вывести число 1538. 

# Задания 2

## Рассматривается множество целых чисел, принадлежащих отрезку [1125; 7856], которые делятся а 3 и не делятся на 7 и 5. Найдите количество таких чисел.

### 
```
s = 0
for i in range (1125, 7856):

        if i % 5 != 0 and i % 7 != 0:
            s += 1
print(s)
```

# Задания 3

## Рассматривается множество целых чисел, принадлежащих отрезку [100; 999], которые являются палиндромом (одинаково читаются как слева направо, так и справа налево). Найдите сумму таких чисел.

### 
```
n = 0
for i in range(100, 1000):
    s = str(i)
    if s == s[::-1]:
        n += i
print(n)
```
## Для самопроверки: программа должна вывести число 49500.

# Задания 7

## В программу вводится число N, после чего - N натуральных чисел, не превышающих 1000. Программа должна найти и вывести на экран максимальное число, кратное 7 и не кратное 9.

### 
```
n = int(input())
s = 0
for i in range(n):
    x = int(input())
    if x % 7 == 0 and x % 9 != 0:
        if x > s:
            s = x
print(s)
```
## Для примера можно ввести: 7, 14, 28, 35, 49

# Задания 8

## В программу вводится число N, после чего - N натуральных чисел, не превышающих 1000. Программа должна найти и вывести на экран минимальное двузначное число, восьмеричная запись которого оканчивается на 3.

### 
```
y = int(input())
n = 1000
for i in range(y):
    x = int(input())
    if 10 <= x <= 99:
        if oct(x)[-1] == 3:
            if x < n:
                n = x
print(n)
```

# Задания 10 

## В программу вводится число N. Требуется определить, является ли это число простым.

### 
```
n = int(input())
is_prime = True
if n < 2:
    is_prime = False
else:
    for i in range(2, n):
        if n % i == 0:
            is_prime = False
            break
if is_prime:
    print("Простое число")
else:
    print("Не простое число")
```
